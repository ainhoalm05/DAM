Ejercicio 1: Validar un formulario con expresiones regulares

Crea una página con un formulario que tenga los siguientes campos: Nombre (solo letras, mínimo 3 caracteres) Email (formato válido) DNI español (8 números seguidos de una letra) Cuando el usuario haga clic en el botón "Validar", debes usar expresiones regulares para comprobar cada campo y mostrar mensajes de error si hay fallos.

Ejercicio 2: Contador de clics usando eventos

Crea una página que tenga un botón con el texto "Haz clic". Cada vez que el usuario haga clic, se debe: Incrementar un contador (visible en pantalla) Mostrar un mensaje cuando se llegue a 10 clics: "¡Has llegado a 10 clics!" Utiliza event listeners correctamente y evita usar atributos onclick en HTML.

Ejercicio 3: Juego de tablero sencillo con DOM

Crea un "tablero" de 3x3 (tipo Tic-Tac-Toe) con divs o botones. Al hacer clic en una celda, debe aparecer una "X" o una "O" alternando cada turno. Usa JavaScript puro para crear el tablero en el DOM. Desactiva las celdas ya marcadas. Muestra un mensaje de quién ha ganado (puedes simplificar la lógica o no implementarla completa si no da tiempo).

Ejercicio 4: Búsqueda dinámica con expresiones regulares

Crea una lista de palabras (por ejemplo: JavaScript, Java, Python, C++, Kotlin). Añade un input de texto arriba. Cada vez que el usuario escriba en el input, muestra solo las palabras que coincidan con lo que escribe usando una expresión regular (por ejemplo, si escribe "ja", que se muestren Java y JavaScript).
