1. Codificador

Codificador de texto.Utiliza un campo input para que el usuario introduzca el texto a codificar, el texto se debe codificar inmediatamente según escribe el usuario. Las letras se codifican con el número de la posición que ocupan en el abecedario español, por ejemplo si el usuario escribe ABC o abc (no diferencia entre mayúsculas y minúsculas) debe aparecer debajo resultado:123
